## 摘要
GraphRAG 是检索增强生成（RAG）的扩展，通过引用图结构数据作为外部知识来支持大型语言模型（LLM）。虽然这种技术理想地捕获了复杂的关系，但它经常在处理LLM的图表示时遇到困难，特别是对于冻结的LLM，这是由于基于图的潜在特征和基于文本的潜在特征之间的不对齐。我们通过引入自适应掩码图嵌入（AGE）来解决这个问题。AGE采用基于掩码的自监督学习（SSL）方法中的Transformer。我们设计了与文本嵌入编码器类似的结构，以解决潜在特征不对齐的问题。与自然语言文本相比，图形是简洁的表示，并且存在持有主导上下文信息的关键节点，这些节点很难从周围环境中预测。掩码这样的关键节点会导致SSL过程的低效。因此，AGE专注于预测除关键节点以外的节点，利用一个可学习的节点采样器。我们的实验结果表明，AGE显著改进了在GraphQA任务中使用非参数搜索组件的方法，在四个具有不同特征的基准数据集上实现了卓越的准确性。

## Abstract
GraphRAG is an extension of retrieval-augmented generation (RAG) that supports large language models (LLMs) by referring to graph-structured data as external knowledge. While this technique ideally captures intricate relationships, it often struggles with graph representations for LLMs, particularly for frozen LLMs, due to the misalignment between graph-based and text-based latent features. We tackle this issue by introducing the {\it Adaptive-masking for Graph Embedding (AGE)}. AGE employs a Transformer in a mask-based self-supervised learning (SSL) approach. We designed the architecture similar to text embedding encoders, addressing the latent feature misalignment. In contrast to natural language texts, graphs are concise representations, and there exist {\it key nodes} that hold dominant contextual information, which are challenging to predict from their surroundings. Masking such key nodes leads to inefficiency in the SSL process. Therefore, AGE focuses on predicting nodes apart from key nodes, utilizing a learnable node sampler. Our experimental results indicate that AGE significantly improves approaches using non-parametric search component in GraphQA tasks, achieving superior accuracy across four benchmark datasets with distinct characteristics.

### 2. 论文提出的方法论

- **核心思想**：设计一个与文本编码器结构类似的 Transformer 编码器（AGE），采用掩码自监督学习（类似 BERT）来学习图节点间的结构化关系。为避免掩码关键节点导致的低效，引入一个基于强化学习（REINFORCE）的可学习节点采样器，**仅掩码辅助节点**，从而迫使模型学习从关键节点推断辅助节点，压缩结构化关系。
- **关键技术细节**：
  - **整体框架**（Fig. 1）：检索 → 子图构建 → 文本化+图嵌入（AGE）→ 冻结 LLM 推理。
  - **AGE 模块**（Fig. 2）：
    - **Graph Encoder**（GNN）提取节点特征 \(h_{in}\)。
    - **Node Sampler**（MHA + 线性层 + softmax）输出每个节点被选为“关键节点”的概率，并通过 RL 训练以最小化辅助节点的重建误差。
    - **Concept Encoder-Decoder**：只编码关键节点（\(h_{key}\)），再与辅助节点的占位符拼接，通过解码器重建所有节点表示 \(h_{out}\)。
    - **Target Encoder**（EMA 更新）对所有节点编码得到 \(h_{target}\)，作为 JEPA 的重建目标。
    - **Graph-structure-based Aggregator** 将 \(h_{out}\) 聚合为单个 token \(\bar{h}_g\)，输入 LLM。
  - **损失函数**（相互独立，无需调权）：
    - \(L_{PT}\)：提示调优损失（优化 Target Encoder、Aggregator、Projector）。
    - \(L_{target}\)：MSE 损失，仅对**辅助节点**计算 \(h_{out}\) 与 sg(\(h_{target}\)) 的距离。
    - \(L_{NS}\)：RL 损失，鼓励难以预测的节点被选为关键节点。
- **算法流程**（训练时）：1) 前向得到 \(h_{in}\)；2) Node Sampler 采样关键节点；3) Concept Encoder-Decoder 重建 \(h_{out}\)；4) 分别计算三个损失并更新对应参数；5) 推理时使用 \(h_{out}\)（而非 \(h_{target}\)）输入 LLM。

### 6. 论文的主要结论与发现

- AGE 在四个数据集上均**显著优于**同等条件下的 G-Retriever 和 AMAR 基线，特别是在 ExplaGraphs 上提升幅度高达 26.72 个百分点（Llama3.2 1B）。
- JEPA 架构优于传统生成式架构（GA），可学习节点采样器优于随机掩码。
- 采样率 \(\rho=0.3\) 为最佳选择，能平衡关键节点筛选与重建质量。
- AGE 与 LoRA 结合后，性能进一步提升，且训练成本远低于 LLM 检索器方法。
- AGE 不仅提升了非参数检索器的表现，甚至还超过了部分使用 GPT-4 的 LLM 检索器方法（特别是在 CWQ 数据集上）。

### 7. 优点（方法或实验设计的亮点）

- **方法亮点**：
  - 创新性地将 JEPA 和 RL 驱动的自适应掩码引入 GraphRAG，避免关键节点被掩码导致的低效。
  - 三个损失函数优化互不重合的参数，无需人工调权，简化训练。
  - 推理时使用 Concept Decoder 的输出 \(h_{out}\)，性能超过直接使用 Target Encoder。
- **实验设计亮点**：
  - 在四个差异性很大的数据集（常识推理、视觉问答、大规模 KGQA）上验证泛化性。
  - 对比了多种 LLM 规模（1B、3B、7B、8B、13B）和架构（Llama、Qwen），证明方法的稳健性。
  - 消融实验全面覆盖了 SSL 架构、掩码策略、采样率、GNN 类型、损失权重等关键因素。
  - 提供了训练/推理成本的具体数据，便于实际部署评估。

### 8. 不足与局限

- **固定采样率**：对所有图使用相同的采样率 \(\rho=0.3\)，未考虑不同图中关键节点密度的变化，可能不是最优。
- **任务范围有限**：仅在 GraphRAG 任务（KGQA 场景）上测试，未在图学习任务（节点分类、链接预测）上验证。
- **模型规模限制**：主要测试了中小型 LLM（<13B），未探索更大模型（如 70B+）的效果（受限于计算资源）。
- **理论保证缺失**：缺乏严格的数学证明说明为何非关键节点重建能提升推理性能，仅通过 ELBO 分析给出直觉解释。
- **对检索质量的依赖**：方法依赖非参数检索器提供的子图质量，若检索器完全失败，AGE 提升有限。
- **跨模态局限性**：仅处理图结构数据，未扩展到其他模态（如图像、表格）。
