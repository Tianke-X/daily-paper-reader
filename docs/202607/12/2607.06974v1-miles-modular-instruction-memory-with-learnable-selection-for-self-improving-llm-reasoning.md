## 摘要
大型语言模型（LLM）通过在测试时进行额外计算，日益提升其推理能力，然而现有的大多数工作将每个问题孤立处理。当问题顺序出现时，跨问题积累可复用的经验可以进一步提升性能。现有的基于记忆的方法要么存储整个解题模板，这些模板对新颖问题泛化能力差，要么使用启发式的步骤级选择，但未针对最终答案的正确性进行优化。学习选择策略需要大规模训练数据和固定的动作空间，使得这些方法不适用于测试时场景，因为记忆会逐步扩展且仅有有限的监督可用。我们提出了MILES（Modular Instruction Memory with LEarnable Selection for self-improving LLM reasoning），一个在现实测试时约束下动态扩展步骤级记忆并应用正确性优化的记忆组合的框架。MILES维护由非对称的子目标嵌入和子指令对组成的模块化记忆单元，每个单元关联一个可学习的选择头。这种记忆结构实现了从粗到细的检索机制：粗粒度层支持记忆扩展并从置信样本中收集用于训练选择头的监督信号，而细粒度层应用已学习的选择头对粗粒度候选进行重新排序，并为不确定样本引导推理。MILES在精度-效率权衡方面始终匹配或超越先前方法，实现更优的表现。大量实验证明了其有效性、鲁棒性和可迁移性。

## Abstract
Large language models (LLMs) increasingly improve their reasoning at test time via additional computation, yet most existing works treat each problem in isolation. When problems arrive sequentially, accumulating reusable experience across them can further improve performance. Existing memory-based methods either store whole-solution templates that generalize poorly to novel problems or use heuristic step-level selection that is not optimized for final-answer correctness. Learning selection policies requires large-scale training data and fixed action spaces, making such approaches unsuitable for test-time settings where memory expands incrementally and only limited supervision is available. We propose MILES (Modular Instruction Memory with LEarnable Selection for self-improving LLM reasoning), a framework that dynamically expands step-wise memory and applies correctness-optimized memory composition under realistic test-time constraints. MILES maintains modular memory units consisting of asymmetric pairs of sub-goal embeddings and sub-instructions, each associated with a learnable selection head. This memory structure enables a coarse-to-fine retrieval mechanism: The coarse level enables memory expansion and collects supervision for training selection heads from confident samples, while the fine stage applies learned selection heads to rerank coarse-level candidates and guide reasoning for uncertain samples. MILES consistently matches or outperforms prior methods while achieving superior accuracy-efficiency tradeoffs. Extensive experiments demonstrate its effectiveness, robustness, and transferability.

- **研究动机**：设计一种测试时可自我改进的框架，使LLM能够从已解决问题的自信轨迹中逐步积累可复用的分步推理经验，并通过可学习的选择机制自动组合这些经验，以提升不确定问题的推理准确性，同时保持LLM参数冻结。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **MILES（Modular Instruction Memory with LEarnable Selection）**：构建模块化指令记忆，每个记忆单元是一个非对称对（子目标嵌入 + 子指令文本），每个单元关联一个轻量级可学习选择头。通过**粗到细的检索机制**实现记忆的动态扩展与组合优化。

### 关键技术细节

- **模块化记忆单元**：每个单元包含  
  - 子目标嵌入（sub-goal embedding）：作为检索键，表示下一步推理步骤的目标。  
  - 子指令（sub-instruction）：自然语言指导，用于条件化生成。  
  - 可学习选择头（selection head）：预测该指令在当前推理状态下的适用性。

- **粗到细检索机制（Two-Layer Selection）**  
  - **Layer 1（粗检索）**：基于子目标嵌入的余弦相似度进行检索，设定有效阈值排除无关单元。若无匹配，则LLM直接生成下一步，相关（子目标，推理步骤）对存储为种子用于后续记忆扩展。  
  - **Layer 2（细粒度选择）**：利用已训练的选择头对候选指令打分。通过构建上下文码书（context codebook，由子目标与上下文条件的聚类中心组成），对当前推理状态提取固定长度特征向量（最大相似度池化），然后由选择头输出适用性得分。  
  - **最终得分**：结合Layer 1相似度与Layer 2得分（式3），选取最高分单元应用。

- **选择头的训练**  
  - 将选择形式化为序列决策问题，在KL正则化下的最优策略具有闭式解（式5），采用二进制交叉熵损失（式6）训练每个单元的一个vs-其余的二分类器。  
  - 正例：应用该指令导致最终答案正确的状态；负例：应用该指令导致错误答案或应用其他指令成功。  
  - 训练数据来自**蒙特卡洛树搜索（MCTS）**在**自信样本**上的展开。

- **测试时推理流程**  
  1. **自信度路由**：对每个测试样本，生成K个独立回答。若多数一致，视为自信样本；否则为不确定样本。  
  2. **自信样本处理**：运行MCTS，使用Layer 1选择（基于子目标相似度）采样指令进行展开，根据伪标签（多数投票答案）计算正确性回报，收集（状态，正确性标签）对。同时，从展开轨迹中聚类并总结新的记忆单元。  
  3. **不确定样本处理**：应用记忆指导的树搜索（类似Tree-BoN），使用Layer 1检索候选指令，Layer 2选择头重排，选取最佳指令生成下一步，逐阶段扩展，最终多数投票确定答案。  
  4. **增量学习**：记忆单元和选择头可逐步添加，无需重新训练已有头。
