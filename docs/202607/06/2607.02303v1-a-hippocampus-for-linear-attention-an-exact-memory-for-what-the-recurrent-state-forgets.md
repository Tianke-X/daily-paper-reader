## 摘要
线性注意力和状态空间语言模型将前缀压缩为固定大小的循环状态，实现O(1)内存，但代价是丢失精确记忆：当许多键-值关联竞争时，早期事实被覆盖，针在干草堆中的检索能力下降。受互补学习系统的启发，我们为线性注意力增添了一个海马体补充。HOLA（海马体线性注意力）保留通常的delta规则状态作为压缩记忆，并添加一个有界精确KV缓存，形成半参数化的测试时记忆：状态建模可线性压缩的结构，而缓存存储不应强行通过该状态的关联。缓存无需学习驱逐模块，通过写入那些具有较大beta * ||e||（即实际提交给状态的预测残差）的令牌；解耦的RMSNorm-gamma缓存读取将这些精确KV对转化为尖锐的检索而非软平均。在150亿个SlimPajama令牌上训练340M参数，HOLA将Wikitext困惑度从27.32降至22.92（-16.1%），低于完全注意力Transformer++（26.88），并将LAMBADA困惑度从30.95降至30.26。它还实现了最佳的线性上下文检索，并在RULER针在干草堆检索中，对于长达32k令牌（训练长度的16倍），其鲁棒性远优于GDN或匹配的HOLA+近因缓存。

## Abstract
Linear-attention and state-space language models compress the prefix into a fixed-size recurrent state, yielding O(1) memory at the cost of a lossy exact memory: when many key--value associations compete, earlier facts are overwritten and needle recall degrades. Inspired by Complementary Learning Systems, we give linear attention a hippocampal complement. HOLA (Hippocampal Linear Attention) keeps the usual delta-rule state as a compressive memory and adds a bounded exact KV cache, forming a semiparametric test-time memory: the state models linearly compressible structure, while the cache stores associations that should not be forced through that state. The cache writes without a learned eviction module, keeping tokens with large beta * ||e||, the prediction residual actually committed to the state; a decoupled RMSNorm-gamma cache read then turns these exact KV pairs into sharp retrieval rather than soft averaging. At 340M parameters trained on 15B SlimPajama tokens, HOLA lowers Wikitext perplexity from 27.32 to 22.92 (-16.1%), below a full-attention Transformer++ (26.88), and improves LAMBADA perplexity from 30.95 to 30.26. It also achieves the best linear in-context retrieval and remains much more robust than GDN or a matched HOLA+recency cache on RULER needle-in-a-haystack recall out to 32k tokens (16x its training length).

- **核心问题**：线性注意力和状态空间模型（如 DeltaNet、Gated DeltaNet）将整个前缀压缩为固定大小的循环状态，虽然实现了 O(1) 内存和次二次时间，但丢失了精确记忆能力。当大量键-值关联竞争时，早期事实会被覆盖，导致针在干草堆检索（needle-in-a-haystack）能力严重下降，尤其远距离精确回忆退化。
- **整体含义**：受认知科学中互补学习系统（CLS）理论的启发——海马体负责快速记录特定新颖事件，新皮层缓慢提取通用结构——该论文为线性注意力模型添加一个“海马体”组件（精确 KV 缓存），用于补偿循环状态丢失的精确关联。这项工作旨在同时获得线性注意力的高效性和全注意力的精确回忆能力。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将记忆读出示为测试时回归问题，线性注意力的循环状态是参数的、可压缩的估计器，而有界精确 KV 缓存是非参数的修正。二者构成半参数测试时记忆（semiparametric test-time memory）。
- **关键技术细节**：
  - **缓存写入（what to store）**：利用 delta 规则本身的写入幅度 `β·||e||` 作为“惊讶度”信号。`e` 是残差（预测误差），`β` 是写入强度。该值越大，说明该令牌对状态的改变越大，越应该被缓存保留。无需额外学习的驱逐模块，缓存自动保留 top-w 个惊讶度最高的令牌，而非最近令牌。
  - **缓存读取（how to read）**：为避免扁平化软平均，对缓存路径的 q、k 使用解耦的 RMSNorm-γ（类似 Qwen3），使得有效 logit 约为 √d·cos（约 11 cos），实现近似 argmax 的锐利检索。状态更新路径仍保持 unit-L2 归一化以保证 delta 规则的稳定性。
  - **整体架构**：基于 Gated DeltaNet（GDN）骨干，每层添加大小固定的精确 KV 缓存（默认 w=64 个持久令牌 + 当前处理块的 C=256 个令牌 + 1 个 null sink）。缓存与状态并行，输出为 `o_t = q⊤St + λ_t g_t(q_t)`，其中 g_t 是缓存上的锐利 softmax 注意力。
- **公式说明**：利用 delta 规则更新方程 `S_t = S_{t-1} + β_t k_t e⊤_t`，写入幅度 `m_t = β_t ||e_t||` 作为缓存驱逐分数。

### 6. 论文的主要结论与发现

- **困惑度**：HOLA 在 Wikitext 上从 GDN 的 27.32 降至 22.92（-16.1%），甚至低于全注意 Transformer++（26.88）。LAMBADA 困惑度也是所有对比模型中最优。
- **上下文检索**：FDA 提升 72%（11.7→20.1），SWDE 提升 24%（29.0→35.9），SQuAD 小幅提升。

- **规模化一致性**：从 46M 到 340M，HOLA 相对 GDN 的困惑度降低幅度稳定在 15-16%。
- **结论**：线性注意力模型自身的更新规则已经“诊断”了它会忘记什么，用少量精确缓存保留那些“惊讶”令牌，可以有效恢复远距离精确回忆，且几乎不增加参数量和推理内存。

### 7. 优点
