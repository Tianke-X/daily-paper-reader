## 摘要
过滤近似最近邻（FANN）搜索是一项关键且新兴的任务，用于增强向量数据库的查询能力，支持推荐系统、检索增强生成（RAG）和智能体记忆等应用。然而，现有大多数方法仅限于范围或标签过滤，通常会导致不可接受的索引构建时间和内存开销。谓词无关的方法进一步难以有效处理各种谓词选择性。在本文中，我们提出了EMA，一种支持混合数值和分类属性上的多谓词查询以及高效动态更新的过滤ANN算法。EMA引入标记作为附加在图边上的紧凑摘要，提供保守的谓词和几何感知引导，在标记级别实现零假阴性。在查询处理过程中，EMA执行标记增强的联合搜索，并带有有界的边恢复机制，从而在保持图可导航性的同时实现高效过滤。大量实验表明，在各种工作负载下，EMA相比最先进的通用过滤ANN方法实现了1.68倍到12.25倍的加速。

## Abstract
Filtering Approximate Nearest Neighbor (FANN) search is a critical and emerging task for strengthening the query capability of vector databases, supporting applications such as recommendation systems, retrieval-augmented generation (RAG), and agent memory. However, most existing methods are limited to range or label filtering, often incurring unacceptable index construction time and memory overhead. Predicate-agnostic approaches further struggle to handle a wide range of predicate selectivities effectively. In this paper, we propose EMA, a filtering ANN algorithm that supports multi-predicate queries over mixed numerical and categorical attributes, and efficient dynamic updates. EMA introduces Markers as compact summaries attached to graph edges, providing conservative predicate- and geometric-aware guidance with zero false negatives at the Marker level. During query processing, EMA performs Marker-augmented joint search with a bounded edge recovery mechanism, enabling efficient filtering while preserving graph navigability. Extensive experiments demonstrate that EMA achieves 1.68x--12.25x speedup over state-of-the-art general filtering ANN methods across diverse workloads.

# EMA: 带有通用属性过滤和动态更新的近似最近邻搜索——论文总结

## 1. 核心问题与研究动机

- **背景**：向量数据库中的近似最近邻（ANN）搜索仅依赖向量距离，无法满足实际应用中同时考虑语义相似性和结构化属性约束（如时间戳、类别标签）的需求。过滤近似最近邻（FANN）搜索应运而生，旨在支持谓语约束下的向量检索。
- **现有方法的不足**：
  - 大部分方法局限于范围过滤或标签过滤，不支持混合数值/分类属性的多谓词组合。
  - 谓语无关方法在低选择性（满足条件的点稀疏）或“off-cluster query”（OCQ，满足条件的点在查询局部稀疏但在远处聚集）场景下鲁棒性差。
  - 多数方法不支持高效的动态更新，索引维护成本高。
- **核心挑战**：缺乏一种统一、选择性鲁棒、支持通用属性过滤且能动态更新的ANN索引方案。

## 2. 提出的方法论：EMA

### 核心思想
EMA基于HNSW两级图结构，在底层图的边上附加紧凑的**Marker**（标记），作为一种边中心的、几何感知的摘要。Marker聚合了RNG剪枝过程中被支配邻居的属性信息，提供**零假阴性**的谓语预检查。查询时，底层搜索通过Marker引导的联合搜索和边界边恢复机制，实现高效过滤并保持图的可导航性。

### 关键技术细节

1. **Codebook（码本）**  
   - 对每个属性，将值域离散化为$s$个桶，生成从原值到比特位的映射。数值属性用等频分桶，分类属性按频率分组。
   - 每个节点的属性被编码为一个$s\times m$位的Marker向量。

2. **Marker增强的RNG剪枝（Algorithm 3）**  
   - 插入节点时，候选邻居按距离排序，采用RNG风格剪枝：若候选$v$被已选邻居$w$支配（$dist(w,v)<dist(u,v)$），则边$(u,v)$被剪掉，但其Marker通过按位或传播到支配边$(u,w)$上。
   - **多样性感知剪枝**：保留至少$M/3$个最近邻后，只有激活了当前计数低于阈值$M_{div}$的桶的邻居才被保留，促进属性多样性。
   - 每个保留的边附加其Marker。

3. **查询时Marker引导的联合搜索**  
   - 查询构造QueryMarker（由谓语编码得到）。
   - 数值属性要求Marker与QueryMarker的对应段按位与非零；分类属性要求完全覆盖查询标签（按位与等于QueryMarker）。
   - 边扩展时，只有通过Marker检查（MCheck）的边才被探索，否则跳过。
   - **边界边恢复**：当通过Marker检查的边数量低于阈值$d_{min}$时，恢复最近的不匹配边以保证最小出度，避免搜索停滞。

4. **动态更新**  
   - 插入：自然支持，增量构造Marker。
   - 删除：惰性标记，当删除比例超阈值（20%）触发批处理补丁（patch），用被删节点的最近有效邻居替换指向它的边；超50%时全重建。
   - 修改：属性修改仅更新邻居反向边的Marker（OR操作）；向量+属性修改采用删除+插入流程。

### 复杂度分析
- 索引构建时间：$O(M \cdot ef_c \cdot n \log n \cdot (M+m))$，与HNSW同量级（$M,m$为常数）。
- 空间复杂度：$O(nM \cdot s m)$，常数因子开销。
- 理论保证：Marker级别零假阴性；分析了两种假阳性来源（支配聚合和码本粒度），并给出控制假阳性率的码本大小充分条件。

## 6. 主要结论与发现

1. EMA在多种谓语类型和选择性下均取得一致的最优QPS，速度提升显著。
