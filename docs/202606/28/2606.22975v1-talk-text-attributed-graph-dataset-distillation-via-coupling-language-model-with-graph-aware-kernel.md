## 摘要
文本属性图（TAG）广泛应用于许多实际领域，学习TAG需要联合建模文本语义和图结构。建模TAG的标准方法是结合语言模型（LM）和图神经网络（GNN），但联合训练计算成本高且难以扩展。数据集蒸馏是一种有前景的减少训练成本的方法，但现有方法不太适用于TAG，因为它们通常针对单模态设计，或者在蒸馏过程中仍需要在完整数据集上反复训练昂贵的LM-GNN模型。为解决此问题，我们提出TaLK，一种有效的TAG数据集蒸馏方法，它将LM与图感知神经正切核耦合。该设计实现了高效的数据集蒸馏，避免了在完整数据集上重复联合训练，同时反映文本和结构信息以有效学习TAG。在多个TAG基准上的实验表明，TaLK始终优于现有基线，并且仅使用1%的合成数据即可达到完整数据集性能的97%。

## Abstract
Text-attributed graphs (TAGs) are widely used in many real-world domains, and learning on TAGs requires jointly modeling text semantics and graph structure. A standard approach for modeling TAGs is to combine a language model (LM) and a graph neural network (GNN), but joint training is computationally expensive and difficult to scale. Dataset distillation is a promising way to reduce training costs, but existing methods are not well suited to TAGs because they are typically designed for a single modality or still require repeatedly training expensive LM-GNN models on the full dataset during distillation. To address this, we propose TaLK, an effective dataset distillation method for TAGs that couples an LM with a graph-aware neural tangent kernel.This design enables efficient dataset distillation, avoiding repeated joint training on the full dataset while reflecting both textual and structural information for effective TAG learning.Experiments on multiple TAG benchmarks show that TaLK consistently outperforms existing baselines and achieves up to 97% of full-dataset performance with only 1% synthetic data.

## 1. 核心问题与整体含义

**研究动机**  
文本属性图（Text-attributed Graph, TAG）在学术网络、社交网络、电商平台等场景广泛存在。有效建模TAG通常需要联合训练语言模型（LM）与图神经网络（GNN），但联合训练计算开销极大——更新LM参数需要保留邻接节点的LM计算中间状态，难以扩展到大规模图。现有数据集蒸馏方法要么针对单模态设计（不适用于TAG），要么在蒸馏过程中仍需反复在完整数据集上训练昂贵的LM-GNN模型，未能解决可扩展性问题。此外，解耦式蒸馏（先蒸馏文本再蒸馏图）虽然可行，但无法同时捕捉文本语义与图结构信息，效果次优。

**整体含义**  
本文提出TaLK方法，首次实现**无需解耦且避免重复全数据训练**的TAG数据集蒸馏，在保持联合训练优势的同时，大幅降低计算和存储开销，使蒸馏后的合成数据集在极小规模下仍能达到接近全数据训练的精度。

## 2. 方法论

**核心思想**  
将LM与**图感知神经正切核（Graph-aware Neural Tangent Kernel）**耦合，在外层优化中用闭核脊回归（Kernel Ridge Regression, KRR）替代可训练的GNN，从而避免在全数据集上重复训练LM-GNN模型。同时引入**批量梯度注入（Batch-wise Gradient Injection）**，使核目标函数在迷你批训练中保持一致。

**关键技术细节**
- **合成数据集初始化**：从全训练集中按类别均衡采样M个节点，取其文本通过LM词嵌入层得到初始token嵌入；合成图结构固定为单位矩阵（A' = I），让合成特征隐式编码结构信息。
- **内层循环（Inner-loop）**：在合成数据集上联合训练LM和GNN，优化交叉熵损失（公式8）。这使得LM参数可通过GNN梯度更新，保持联合训练对齐。
- **外层循环（Outer-loop）**：
  - 使用内层训练后的LM对全数据集和合成数据集提取表示 \(H_T\) 和 \(H_S\)。
  - 采用结构神经正切核（SNTK）计算核矩阵 \(K_{SS}\) 和交叉核矩阵 \(K_{TS}\)，反映图结构信息。
  - 通过KRR获得合成数据集对全数据集的标签预测 \(\hat{Y} = K_{TS}(K_{SS}+\lambda I)^{-1}Y'\)（公式12）。
  - 外层损失为 \(\mathcal{L}_{KRR} = \frac{1}{2}\|Y_L - \hat{Y}_L\|_F^2\)（公式13），优化合成特征 \(X'\)。
- **批量梯度注入**：将合成数据集分成多个batch，先计算全局表示 \(H_S\) 和全局损失，然后从全局梯度中取出各batch对应切片 \(G^{(b)}\)，再反向传播通过LM得到各batch的合成特征梯度，从而保持原外层优化目标不变（公式14-15）。
- **内存优化**：对SNTK采用高效邻域聚合（避免构造Kronecker乘积）和基于特征的归一化近似，降低内存占用。

**算法流程**（算法1）  
初始化合成数据集 \(S_0\) → 重复K轮：① 内层：在\(S_k\)上联合训练LM和GNN（T步）；② 外层：计算KRR损失，用批量梯度注入更新合成特征（M步）；③ 输出最终合成数据集 \(S_K\)。

1. **性能优异**：TaLK在所有数据集和比例下一致优于所有直接竞争对手（核集选择和解耦蒸馏），在11/12个配置下甚至超过依赖全文本的图蒸馏方法。

5. **各组件必要性**：图感知核和批量梯度注入对性能和内存效率均至关重要。

## 7. 优点

- **首次实现TAG端到端联合蒸馏**：无需解耦文本和图，保留两者交互信息，提升合成数据质量。
